# Contributing

Issues and discussion are welcome.

Outside pull requests are temporarily held while the project decides between a
Developer Certificate of Origin and a Contributor License Agreement. Please do
not submit code until the contribution policy is published. This avoids
accepting contributions under ambiguous terms.

Security reports must follow [SECURITY.md](SECURITY.md), not a public issue.
