# Nedyl

Nedyl is an open-source voice dictation system for Windows and Android, with
bring-your-own-key, self-hosted, and hosted-cloud modes.

This repository is an early scaffold. Product code will live in:

- `backend/` — hosted and self-hosted API
- `desktop/` — Windows client based on Handy
- `android/` — Android IME based on Dictate
- `eval/` — public dictation quality evaluation
- `docs/` — product and self-hosting documentation
